# 错误码

## 登录主机

| 错误码 | 说明                       |
| :----- | :------------------------- |
| 0      | 正常，登录成功             |
| 1      | 用户名或密码错误           |
| 2      | 客户端和中控版本不匹配     |
| 3      | 登录报文结构错误           |
| 4      | 用户冲突（已弃用）         |
| 5      | 中控系统错误               |
| 6      | 中控未被认证               |
| 7      | 登录账号无主机（已弃用）   |
| 8      | 主机不在线（仅限远程访问） |
