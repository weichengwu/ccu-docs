# 情景模式类型

情景模式类型是中控预设的，可以随意设置一个值，对功能没有影响。

| 类型 | 含义         |
| :--- | :----------- |
| 1    | 聚会模式     |
| 2    | 会议模式     |
| 3    | 家庭影院     |
| 4    | 回家模式     |
| 5    | 烛光模式     |
| 6    | 离家模式     |
| 7    | 早晨模式     |
| 8    | 睡觉模式     |
| 9    | 影院散场模式 |
| 10   | （保留）     |
| 11   | 机器人       |
| 12   | 私车入库     |
| 13   | 音乐派对     |
| 100  | 自定义       |
