网络链路¶

中控发现链路¶

客户端发现局域网内中控主机的链路，除此之外全部通过中控服务链路传输数据。

  • 通信方式：UDP 广播

  • 通信端口：8989

中控服务链路¶

客户端与中控主机进行数据交互的链路。

  • 通信方式：TCP

  • 通信端口：5000

TCP 粘包解决方案：根据标志进行区分，! 表示开始，$ 表示结束。
