主机状态¶

准确的说是客户端与主机之间的连接状态，主要是通知客户端 UI 层，及时作出响应提示用户。下面是推荐的设计，也可以不完全参考。

初始化¶

客户端刚刚初始化

连接中¶

客户端正在与主机建立 TCP 连接

登录中¶

正在登录主机

登录成功¶

成功登录主机，此时即可和主机进行一切通讯，不过建议优先同步主机内的数据

同步中¶

此时正在同步主机内的数据

同步完成¶

此时数据已同步完毕，同步报文内的数据已存入本地数据库，上层拿到通知后即可调用数据库查询接口，查询需要的数据。

断开连接¶

客户端与主机的 TCP 连接断开，在监控到断开连接的状态时，需要及时向用户反馈，同时尝试重新建立连接。
